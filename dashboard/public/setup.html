Quick tips

  • Ollama: run ollama serve, then ollama pull qwen3:14b (or your preferred models).
  • LM Studio: load a model → start Local Server (usually port 1234) → pick provider “LM Studio”.
  • Paper server: Start-MCAI.cmd or scripts\Start-Server.ps1, join 127.0.0.1:25565 on Java matching the configured version.
  • Bot: cd bot → npm start. Restart the bot after saving LLM settings so all modules reload config.
  • Dashboard token defaults to change-me-local-token in local testing.
